# 💱 ValutaGo

ValutaGo adalah website konversi mata uang (currency converter) yang memungkinkan pengguna menghitung nilai tukar antar berbagai mata uang dunia secara cepat dan praktis.

---

## 🌐 Live Demo

👉 [https://anandadavidsaputra.github.io/ValutaGo/](https://anandadavidsaputra.github.io/ValutaGo/)

---

## 🧩 Overview

ValutaGo dibangun sebagai proyek latihan pengembangan web untuk:

* mengonversi nilai mata uang antar negara
* menampilkan hasil konversi secara instan
* menampilkan informasi nilai tukar (exchange rate)
* melatih implementasi perhitungan berbasis JavaScript

Website ini memiliki antarmuka sederhana dengan dua pilihan mata uang (from–to), input jumlah nominal, serta hasil konversi yang ditampilkan secara dinamis.

---

## ✨ Fitur Utama

* Input nominal yang fleksibel
* Dropdown pilihan mata uang
* Perhitungan otomatis
* Tampilan hasil konversi
* Informasi kurs yang digunakan
